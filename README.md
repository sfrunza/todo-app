# Todo app

Simple Rails Todo app with due dates.(minimal styling).

## Installation

Ruby 3.0.0,  
Rails 6.1.4

```bash
git clone https://github.com/sfrunza/todo-app.git
cd todo-app
bundle install
rake db:create && rake db:migrate
rails s

```
